## Course Summary

- Two-day live course for intermediate Python developers
- Focused on building Python projects with Codex CLI inside the codebase
- Course dates: July 11-12, 2026
- Each live session runs for four hours via Zoom
- Instructor: Philipp Acsany

## Course Focus

- Scaffold a complete Python CLI project from an empty directory
- Use a repeatable plan, implement, verify, and ship workflow
- Work with uv, Git, GitHub, tests, Click, and Textual
- Create reusable Codex CLI skills
- Debug AI-generated code using logs and reproducible failures
- Refactor shared business logic and build a terminal dashboard

## Preparation

The course expects familiarity with Python fundamentals, modules, packages,
virtual environments, terminal usage, and basic Git/GitHub workflows. Before
the course, learners should have Codex CLI, ChatGPT Plus or Pro, Python with
uv, and the GitHub CLI installed and configured.

## Curriculum

### Day 1 - From Zero to Working Project

- Orient to Codex CLI
- Scaffold a project from a prompt
- Manage Git and GitHub with the agent
- Set project conventions in AGENTS.md
- Build and ship a first feature

### Day 2 - Skills, Debug, Build and Ship

- Create reusable skills
- Investigate and fix bugs
- Build a Textual terminal dashboard
- Explore hooks, MCP, and headless mode
