A tiny command-line second brain

Think it. Save it. Find it.

second_brain turns quick thoughts into plain Markdown files, then gives you two tiny tools to browse and read them again.

01 · CLI command deck

Three moves, no ceremony.

This is the same command set you see in second_brain --help. Click a card to see its example.

second_brain new "My brilliant idea about caching" 
$ second_brain --help

Usage: second_brain [OPTIONS] COMMAND [ARGS]...

  Create and read plain Markdown notes.

Options:
  --help  Show this message and exit.

Commands:
  list  List the configured notes directory and its Markdown notes.
  new   Save THOUGHT as a Markdown note.
  show  Print note NUMBER from the list.
